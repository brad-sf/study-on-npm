# NPM Training Notes

## Install
- `npm i` === `npm install`
- `npm i -S` === `npm i --save`
- `npm i -D` === `npm in --save-dev`

## Scripts
- `npm start`
- `npm test`
- `npm run <taskname>`

## Help
- `npm -h` // command usage
- `npm <command> -h`
- `npm help <command>` // documentation
- `npm help-search <command>`
- `npm api-help` // When using as an API
